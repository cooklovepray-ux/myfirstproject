# Overview  
StayFollowup은 국내 숙소 호스트를 위한 예약 자동 메시지 발송 및 달력 관리 서비스입니다. 네이버예약, 전화, 메신저 등 Airbnb 외 채널로 예약을 받는 소규모 숙소 호스트를 대상으로 하며, 예약 안내 업무 자동화, 오류 및 누락 방지, 운영 효율 향상을 목표로 합니다.

# Core Features  
## 1. 예약 자동 안내 메시지 발송
- 체크인/체크아웃/리뷰 요청 메시지 자동 발송
- 예약이 저장되면 자동으로 체크인 안내 메시지 발송
- 알림톡 + SMS 채널 지원
- 한국어 UI 및 알림 템플릿만 지원

## 2. 멀티 숙소 달력 통합 관리
- 여러 숙소의 예약 현황을 한 달력에서 통합 관리
- 월별 달력 뷰로 전체 일정 파악
- 예약 상세 정보 확인 및 수정/취소 기능

## 3. 취소·짜투리 기간 Flash-Sale 알림 발송
- 빈방(취소) 발생 시 구독자에게 즉시 알림 발송
- 빈날 표시 및 "알림 보내기" 기능
- 발송 기록 관리

# User Experience  
## User Personas
- **타겟 사용자**: 네이버예약/전화/메신저 등 Airbnb 외 채널로 예약 받는 소규모 숙소 호스트

## Key User Stories
1. **[호스트]** 예약이 저장되면 자동으로 체크인 안내 메시지가 발송되길 바란다.

3. **[호스트]** 빈방(취소) 발생 시 구독자에게 즉시 알림을 보내 매출을 유지하고 싶다.

# Technical Architecture  
## System Components
- **Frontend**: React + Vite + TypeScript
- **UI Framework**: shadcn/ui + TailwindCSS (모바일 우선)
- **Backend/Database**: Supabase (인증, 데이터베이스)
- **Messaging Service**: Solapi (SMS/카카오 알림톡)
- **Deployment**: Vercel

## Data Models
- 사용자 계정 (소셜 로그인: 카카오/네이버)
- 숙소 정보 (멀티 숙소 지원)
- 예약 정보 (이름, 전화번호, 입실/퇴실 날짜)
- 알림 설정 (자동 on/off, 채널 선택, 템플릿)
- 구독자 정보 (Flash-Sale 대상 전화번호 리스트, 수신동의)
- 알림 발송 로그

## APIs and Integrations
- Supabase Auth (카카오/네이버 소셜 로그인)
- Supabase Database (예약, 숙소, 알림 설정 등)
- Solapi API (SMS/카카오 알림톡 발송)

## Infrastructure Requirements
- Vercel 배포 환경
- Supabase 프로젝트 설정
- Solapi 계정 및 API 키

# Risks and Mitigations  
## Technical Challenges
- **Solapi API 연동 복잡도**: Solapi 공식 문서 참고 및 테스트 계정으로 충분한 테스트 진행
- **실시간 알림 발송 타이밍**: Supabase Functions 또는 클라이언트 사이드 스케줄링 고려
- **멀티 숙소 데이터 관리**: Supabase RLS(Row Level Security) 정책으로 데이터 격리 보장

## MVP Scope Management
- **제외 기능 명확화**: 후기/별점, 투숙객 페이지, 다국어, OTA 연동은 MVP에서 제외
- **Mobile-First 우선**: PC 전용 UI는 반응형만 지원, 우선순위 낮음
- **한국어만 지원**: 다국어 기능은 MVP 이후로 연기

## Resource Constraints
- **Solapi 비용**: 무료 티어 또는 테스트 계정으로 초기 개발 진행
- **Supabase 무료 티어 제한**: 데이터베이스 쿼리 및 저장 공간 모니터링

# Success Metrics (KPI)
1. 1호스트 운영 테스트 성공

## 화면 설계 상세
### (A) 로그인 전 — 랜딩 페이지
- 헤더: 로고 / 기능 소개 / 요금제 / 로그인 버튼
- 히어로: 핵심 메시지 + CTA(Button: 시작하기)
- 기능 소개: 예약 자동화 / 달력 통합관리 / Flash-Sale
- 후기 섹션: 실제 숙소 호스트 간단 의견
- 요금제: 월 구독형 가격 3단계 (Starter/Pro/Premium)
- 푸터: 개인정보처리방침, 이용약관, 이메일 문의

### (B) 로그인 후 — 앱 내부
- 숙소 선택 화면: 숙소 리스트 + "숙소 추가"
- 대시보드(숙소별): 예약 요약, 다음 체크인 알림, 빠른 등록 버튼
- 달력 뷰: 월별 달력, 예약 상세, 수정/취소
- 예약 등록: 이름/전화번호/입실/퇴실 날짜
- 알림 설정 & 로그: 자동 on/off, 채널 선택, 로그 조회, 템플릿 편집
- 구독자 관리: 전화번호 리스트 + 수신동의 on/off
- Flash-Sale 발송: 빈날 표시 + "알림 보내기" + 발송 기록
- 설정 & 계정: 로그인/구독/숙소정보 관리
